# Add-ons

Some accessory designs to use alongside the Lightbeam.  These are all designed to be used with M6 nuts and bolts.

* [BaseClamp.FCStd](BaseClamp.FCStd) - a 3D printed clamp to go round the wider-diameter end tubes
* [TubeClamp.FCStd](TubeClamp.FCStd) - a 3D printed clamp to go round the frosted tube of the lightbeam
* [BaseSleeve.FCStd](BaseSleeve.FCStd) - a 3D printed sleeve to allow the Lightbeam to be fitted to a rubber base to turn it into a cycling bollard
